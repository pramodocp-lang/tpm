EDUCATIONAL CRM SYSTEM - FULL TPM PROJECT PLAN
==============================================

1. PROJECT OVERVIEW
===================

Project Name:
Educational CRM System Enhancement Project

Project Type:
Internal Product / Platform Enhancement

Objective:
Build a centralized CRM system to manage the student lifecycle,
improve reporting accuracy, and provide role-based dashboards
for Head Office (HO), Branch Offices, Counsellors, Front Office,
and Mentors.

------------------------------------------------------------

2. PROBLEM STATEMENT
====================

Current CRM system is fragmented across teams, resulting in:

- Manual reporting across branches
- No single source of truth for student data
- Limited visibility for Head Office (HO)
- Delayed decision-making
- Inefficient lead tracking and follow-up

------------------------------------------------------------

3. PROJECT GOALS
================

Business Goals:
---------------
- Improve student lifecycle tracking efficiency by 40%
- Increase lead conversion visibility by 30%
- Reduce reporting effort by 50%
- Improve decision-making speed for HO

Technical Goals:
----------------
- Build a centralized CRM platform
- Implement role-based access control (RBAC)
- Create real-time dashboards
- Ensure scalable and secure architecture
