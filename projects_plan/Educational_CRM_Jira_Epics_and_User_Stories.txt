=========================================================
EPIC 1: LEAD MANAGEMENT
=========================================================

User Story 1
Title: Create Lead

As a Counsellor,
I want to create a new student lead,
So that I can track potential admissions.

Acceptance Criteria:
- Lead name can be added.
- Contact details can be added.
- Course interest can be selected.
- Lead status defaults to New.

---------------------------------------------------------

User Story 2
Title: Update Lead Status

As a Counsellor,
I want to update lead status,
So that I can track lead progress.

Acceptance Criteria:
- Status options available.
- Changes are saved.
- History is maintained.

---------------------------------------------------------

User Story 3
Title: Search Leads

As a Counsellor,
I want to search leads,
So that I can quickly find student records.

Acceptance Criteria:
- Search by name.
- Search by mobile number.
- Search by email.
